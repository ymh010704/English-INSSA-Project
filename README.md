# English-INSSA-Project

전종설(2) 공동 작업

# 1) 우선 해야할 작업

`개인 로컬 폴더에서 git clone 하시구  push는 main 말고 각자 작업 공간에 해주세요!
  아래 따라 하시면 됩니다.
`

##### 현재 AWS EC2 웹서버 작동 확인 완료 + docker 기본 틀 잡기 완료

```bash
$ git clone https://github.com/ymh010704/English-INSSA-Project
$ cd English-INSSA-Project
$ cp .env.example .env
$ docker compose up -d
```
